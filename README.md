# Transaction Tracker PWA

This project has been converted into a Progressive Web App (PWA).

## Changes Made

- **`dashboard/manifest.json`**: Added the web app manifest to define the application's name, icons, and other properties.
- **`dashboard/sw.js`**: Created a service worker to cache application assets and enable offline functionality.
- **`dashboard/index.html`**: Updated to link the manifest and register the service worker.

## How to Install

1. Open the application in a supported browser (e.g., Chrome, Edge, Firefox).
2. Look for an install icon in the address bar or a prompt to add the app to your home screen.
3. Follow the on-screen instructions to install the PWA on your device.
